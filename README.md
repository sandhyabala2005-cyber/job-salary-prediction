# 💼 Job Salary Prediction

## 📌 Project Overview
This project predicts the salary category (High / Low) based on job-related features using Machine Learning.

## Objective
To build a classification model that predicts whether a job offers a high or low salary based on inputs like experience, education, skills, etc.

## 📊 Dataset
The dataset contains the following features:
- experience_years
- education_level
- skills_count
- company_size
- remote_work
- certifications
- salary (target variable)

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## 🔍 Steps Performed
1. Data Collection  
2. Data Preprocessing  
3. Handling Missing Values  
4. Encoding Categorical Data  
5. Train-Test Split  
6. Model Training (Logistic Regression)  
7. Model Evaluation  

## 🤖 Model Used
- Logistic Regression

## 📈 Model Performance
Accuracy: ~80% (may vary based on data)

## Future Improvements
- Use advanced models (Random Forest, XGBoost)
- Improve accuracy with feature engineering
- Deploy as a web application


## 🙌 Conclusion
This project demonstrates how machine learning can be used to predict salary categories based on job-related features.
